# 🚀 DQN CartPole Solver (PyTorch)

This project implements a Deep Q-Network (DQN) from scratch using PyTorch to solve the CartPole-v1 environment from OpenAI Gym.

---

## 📌 Project Overview

The goal is to train an agent to balance a pole on a moving cart using Reinforcement Learning.

The agent learns using:
- Deep Q-Network (DQN)
- Experience Replay Buffer
- Target Network
- Epsilon-Greedy Exploration

---

## 🎯 Results

- Successfully solved CartPole-v1
- Achieved average reward ≥ 195 over 100 episodes

---

## 🧠 Key Concepts

- Q-Learning
- Deep Neural Networks
- Exploration vs Exploitation
- Bellman Equation
